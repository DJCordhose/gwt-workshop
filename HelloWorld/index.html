Installation

  • JDK
  • Eclipse JEE
  • Google Eclipse Plugin
  • GWT SDK (Compiler, jars, etc.)
  • Firefox
  • GWT-Dev-Mode-Plugin für Firefox (wird beim ersten Gebrauch automatisch installiert)

Übung: Hello World

Teil 1

  • Installiere die benötigte Software
  • Erzeuge ein GWT-Projekt und lasse dir Sample-Code erzeugen
    • Trage dabei dein SDK in das Google Plugin ein
  • Starte das Projekt im "Development Mode" und probiere es mit Firefox aus
  • Vereinfache den Code des Projekts, so dass nur noch ein einfaches Hello World übrig bleibt - lösche allen Server-Code

Teil 2

  • Wo ist die Modul-Beschreibung und was steht darin?
  • Compiliere das Projekt und lass es im "Production Mode" laufen
    • Fühlt sich das anders an?
    • Wo liegt das JavaScript? Wie wird es geladen?
    • Wie wechselt man zwischen Development- und Production-Mode?
