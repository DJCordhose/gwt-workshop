Zwei Teile
----------

Erster Teil:
zwei Tage: Einführung der absolut notwendigen Grundlagen. Wenig bis keine Folien, gearbeitet wird mit der Originaldoku für GWT, die im Web liegt.
Dabei entwickeln die Teilnehmer zusammen oder mit anderen Teilnehmern eine eigene Applikation. Welche ist ihnen überlassen.

Zweiter Teil:
Es werden an 3 Tagen Projekte als Themenkomplexe vorgestellt und die Teilnehmer setzen sich in Projektform damit auseinander. Dabei bauen Sie die Ideen der Beispielprojekte in ihre eigenen Projekte ein.
- Hier wird noch selbständiger gearbeitet
- Teilnehmer werden aktiv aufgefordert, weiter an ihren eigenen Anwendungen zu basteln

Ein Projekt pro Tag, arbeit an eigenem Thema nach belieben. Ich gebe

GWT Fundamentals (2 days)

Learn to use widgets and layouts provided by GWT to build a simple application typically required in a business environment. Includes different views, styling with CSS, using additional Widgets through plug-ins, handling events and accessing data through RPC.

Topics covered

Creating views
Introducing available widgets
Creating a simple composite widget
Page layout
Creating a view with Java, CSS
Switching between views
Server access
GWT RPC
Navigation
Simple events
Browser history
Using plug-ins
Additional widget types
ClientResources
CSS, images
Security aspects
Deployment


Requirements

Basic Java skills are required
The program includes hands-on exercises so it's essential for you to bring your own laptop.


What you get

The course is divided 50/50 between lectures and hands-on exercises
Good knowledge of GWT
Book of Vaadin
Lunch and refreshments


Please note that the course and all of its materials are in English. Welcome to the GWT Training!
